Extends JSON API with an additional response object called "content_unfiltered". Requires JSON API plugin to first be installed.

== Installation ==

1. Install <a href="http://wordpress.org/plugins/json-api/">JSON API</a> plugin
1. Upload `json-api-unfiltered-content` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
